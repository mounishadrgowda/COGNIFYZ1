# Cognifyz Web Development Internship Tasks

This repository contains the completed web development assignment tasks completed during my internship program at Cognifyz Technologies. The projects range from foundational HTML/CSS implementations to dynamic JavaScript logic and responsive frontend design layouts.

---

## 📂 Project Structure & Task Breakdown

### 🔹 Level 1: Foundational Web Elements
*   **Task 1: Basic Webpage & Navigation** (`index.html` & `style.css`)
    *   Designed a simple semantic webpage featuring structured headings, paragraph containers, and embedded layout images.
    *   Implemented a simple anchor link navigation menu bar and a single-input data entry submit form.
*   **Task 2: Interactive JavaScript Elements** (`Task2.html`)
    *   Engineered an active background event button that randomizes hex-code color strings upon user clicks.
    *   Configured a dynamic `window.onload` alert sequence greeting users based on their localized device time.
    *   Built a pure JavaScript client-side addition calculator tool.

### 🔹 Level 2: Responsive Frontend Architectures
*   **Task 1: Front-end Frameworks (Bootstrap Layouts)** (`level2_task1.html`)
    *   Constructed a responsive, grid-aligned card layout using the Bootstrap CDN utility system.
    *   Organized multiple information modules that seamlessly fluid-shift columns depending on browser viewing breakpoints.
*   **Task 2: Custom Responsive Layout & Media Queries** (`level2_task2.html`)
    *   Developed a native grid-container wireframe using raw CSS media queries without reliance on third-party frameworks.
    *   Created a responsive navigation menu that collapses into an active mobile hamburger menu utilizing pure HTML/CSS checkbox selection logic.

### 🔹 Level 3: Advanced UI Engineering & Capstones
*   **Task 1: Image Manipulation Components** (`level3_task1.html`)
    *   Engineered a pure CSS automatic fading carousel image slideshow.
    *   Built an interactive thumbnail media photo gallery utilizing a fluid grid framework with popup focus overlays.
*   **Task 2: Comprehensive Internship Landing Page** (`level3_task2.html`)
    *   Assembled a full-scale landing platform presenting internship programmatic pathways, feature highlights, qualification requirements, and inquiries channels.

---

## 🛠️ Built Using
*   **Markup & Layout:** HTML5, CSS3, Flexbox, CSS Grid
*   **Interactivity:** Vanilla JavaScript (ES6+)
*   **Frameworks:** Bootstrap 5 Framework Extension
